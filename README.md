# 🍳 Recetas App - Frontend

Interfaz de usuario para la plataforma de Recetas App, una aplicación web SPA (Single Page Application) que permite a los usuarios descubrir, crear y gestionar sus recetas favoritas. 

**Contexto Académico:** Este proyecto fue desarrollado con fines estrictamente académicos como Trabajo Práctico Final para la asignatura Metodologia de Desarrollo Web, de la carrera de Ingeniería en Sistemas dicata en la Universidad Abierta Interamericana (UAI).

## 🚀 Tecnologías Utilizadas

Este proyecto fue inicializado con Vite y construido utilizando el siguiente stack tecnológico:

* **Core:** React 18 + TypeScript
* **Estilos:** Tailwind CSS
* **Enrutamiento:** React Router DOM (Manejo de rutas públicas y privadas)
* **Estado Global:** Redux Toolkit (Gestión de sesión y tokens)
* **Autenticación:** Firebase Auth (Registro y Login con Google)
* **Peticiones HTTP:** Axios
